Not Legal RP Wiki — сворачиваемые сохранённые статьи и проверка загрузки фото

1. Скопируйте папку src в корень проекта с заменой файлов.
2. Новые зависимости не добавлены.
3. Выполните npm run build.
4. Для хостинга Vercel подключите публичное Blob Storage к Production.
5. В Environment Variables должна быть BLOB_READ_WRITE_TOKEN.
6. После добавления Blob или переменной выполните новый деплой.

В /admin/articles появился индикатор состояния хранилища и кнопка «Проверить».
Локально изображения сохраняются в public/uploads/wiki.
На Vercel изображения сохраняются в Vercel Blob.
