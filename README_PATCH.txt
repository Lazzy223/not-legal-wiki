Патч для динамических Wiki-статей Not Legal RP.

Добавлено:
- сохранение статьи в localStorage для всех посетителей;
- раздел «Сохранённые» внутри F10-навигации;
- кнопки «Предыдущая» и «Следующая» только внутри текущей категории;
- учёт каждого открытия статьи, без требования авторизации.

Установка:
1. Скопируйте папку src в корень проекта с заменой файлов.
2. Выполните npm run build.
3. Отправьте изменения в GitHub.
